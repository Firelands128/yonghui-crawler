### Crawler practice for extracting yonghui supermarket information

#### Dependencies
```text
- urllib3
- beautifulsoup4
- validators
- pandas
- openpyxl
```

#### Create Virtual Environment (Optional)
```shell script
pip install virtualenv
virtualenv venv
source ./venv/bin/activate
```

#### Install Dependencies
```shell script
pip install -r requirements.txt
```

#### Run
```shell script
python main.py
```
